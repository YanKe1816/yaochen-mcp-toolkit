# Search Lite MCP

## Overview

`search-lite-mcp` is a modular tool designed to perform lightweight searches using the DuckDuckGo API. It is part of the Yaochen MCP Toolkit, providing reusable search functionality for AI-driven workflows.

## Features

- Lightweight search handler using DuckDuckGo API
- Modular design, easy to integrate into AI agent ecosystems
- Supports automation of tasks and enhances AI capabilities

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/yaochen-mcp-toolkit.git

2.Navigate to the search-lite-mcp folder:
cd search-lite-mcp

3.Install dependencies:
npm install

4.Usage example:
const searchHandler = require('search-lite-mcp');
searchHandler.search('your query');
